		**Python怎么运算？**

			* Python里面怎么做简单的加减乘除
			* 怎样方便的把计算结果保存下来，方便下次计算？
			* Python中小数是怎么计算的呢？小数和整数的混合运算计算结果是怎样的呢？
			* Python中总共有多少数值类型？

		**Python的运算**
			* 运算符:``+、-、*、/、//、%、**``
			* python做简单的加减乘除在解释器里面直输入即可

			>>> 1+1   # 加 1+1输出2
			2 
			>>> 5-2   # 减 5-2输出3 6-8输出-2
			3
			>>> 2*3   # 乘 2*3输出6
			6
			>>> 10/2  # 除 10/2输出5
			5.0
			>>> 10//3 # 取整 10//3输出3或10.0//3.0输出3.0
			3
			>>> 6 % 4 # 取余 10/2输出5
			2
			>>> 2**3  # 幂 2**3输出8
			8
