# 第二章 运算符

## 1 算数运算符

### 1.1 `+`
**加** - 两个对象相加
### 1.2 `-`
**减** - 得到负数或是一个数减去另一个数
### 1.3 `*`
**乘** - 两个数相乘或是返回一个被重复若干次的字符串
### 1.4 `/`
**除** - x 除以 y
### 1.5 `%`
**取模** - 返回除法的余数
### 1.6 `**`
**幂** - 返回x的y次幂
### 1.7 `//`
**取整除** - 向下取接近除数的整数
