# 第二章 运算符

## 1 算数运算符

### 1.1 加:`+`
**加** - 两个对象相加
假设变量a为10，变量b为21

加运算例
```python
a = 21
b = 10

c = a + b
print(c)

# 输出结果 31
```

### 1.2 减:`-`
**减** - 得到负数或是一个数减去另一个数

减运算例
```python
a = 21
b = 10

c = a - b
print(c)

# 输出结果 -11
```

### 1.3 乘:`*`
**乘** - 两个数相乘或是返回一个被重复若干次的字符串

乘运算例
```python
a = 21
b = 10

c = a * b
print(c)

# 输出结果 210
```

### 1.4 除:`/`
**除** - x 除以 y

除运算例
```python
a = 21
b = 10

c = a / b
print(c)

# 输出结果 2.1
```

### 1.5 取模:`%`
**取模** - 返回除法的余数

取模例
```python
a = 21
b = 10

c = a % b
print(c)

# 输出结果 1
```

### 1.6 幂:`**`
**幂** - 返回x的y次幂

幂运算例
```python
a = 21
b = 10

c = a ** b
print(c)

# 输出结果 16679880978201
```

### 1.7 取整除:`//`
**取整除** - 向下取接近除数的整数

取整除例
```python
a = 21
b = 10

c = a // b
print(c)

# 输出结果 2
```

## 2 比较运算符

### 2.1 等于:`==`
**等于** -比较对象是否相等

等于例
```python
a = 21
b = 10

print(a==b)

# a对象与b对象相等返回True 不相等返回false
```

### 2.2 不等于:`!=`
**不等于** - 比较两个对象是否不相等

不相等例
```python
a = 21
b = 10

print(a!=b)

# a对象与b对象不相等返回True 相等返回false
```

### 2.3 大于:`>`
**大于** - 返回x是否大于y

大于例
```python
a = 21
b = 10

print(a>b)

# a大于b成立返回true 不成立返回false
```

### 2.4 小于:`<`
**小于** - 返回x是否小于y。所有比较运算符返回1表示真，返回0表示假。这分别与特殊的变量True和False等价。注意，这些变量名的大写。

小于例
```python
a = 21
b = 10

print(a<b)

# a小于b成立返回true 不成立返回false
```
### 2.5 大于等于:`>=`
**大于等于** - 返回x是否大于等于y。

大于等于例
```python
a = 21
b = 10

print(a>=b)

# a大于或等于b成立返回true 不成立返回false
```

### 2.6 小于等于:`<=`
**小于等于** - 返回x是否小于等于y。

小于等于例
```python
a = 21
b = 10

print(a<=b)

# a小于或等于b成立返回true 不成立返回false
```
